# 8-Month Machine Learning Roadmap (Practical & Structured)

# 🟢 First 5 Months — Foundations + Data

## Month 1-2 — Python & Core Tools

### 💻 Programming
- Python (strong basics up to classes)
- Functions
- OOP basics
- File handling
- Writing clean code

### 🧰 Tools
- Git / GitHub
- Linux basics
- Jupyter Notebook / Colab
- Debugging basics

---

## Month 3–4-5 — Data Analysis & Data Preparation

### 📦 Core Libraries
- NumPy
- Pandas
- Matplotlib
- Seaborn

### 📊 Data Analysis Skills
- Loading CSV/Excel files
- Data inspection (`head()`, `info()`, `describe()`)
- Handling missing values
- Removing duplicates
- Filtering & grouping
- Basic visualization

### 🧹 Data Preparation (Very Important)
- Train/test split
- Feature scaling (Standardization, Normalization)
- Encoding categorical variables
- Outlier detection
- Basic feature engineering

### 🗄 Data Collection Basics
- Reading from CSV
- Basic SQL queries
- Using simple APIs

🎯 Goal by End of Month 3:
You can take a messy dataset and prepare it properly for ML.

## 📚 ML Theory (Concepts First)

- What is Machine Learning?
- Supervised vs Unsupervised learning
- Overfitting vs Underfitting
- Bias–Variance tradeoff
- Train/Test split
- Cross-validation
- Evaluation metrics:
  - Accuracy
  - Precision / Recall
  - F1-score
  - ROC-AUC
  - MSE / RMSE

### Algorithms to Learn:
- Linear Regression
- Logistic Regression
- K-Nearest Neighbors
- Decision Trees
- Random Forest
- Support Vector Machines
- K-Means Clustering
